# 딥러닝과 인공신경망

## 1. 인공신경망 (Artificial Neural Network)

### 정의
- 인간 두뇌의 신경 구조를 모방한 기계학습 모델

### 구조
1. 입력 레이어 (Input Layer)
   - 외부 데이터 수신
2. 히든 레이어 (Hidden Layer)
   - 복잡한 특징 학습
3. 출력 레이어 (Output Layer)
   - 최종 결과 제공

## 2. 딥러닝 (Deep Learning)

### 정의
- 다중 히든 레이어를 가진 인공신경망 사용

### 특징
- 대량 데이터 처리
- 복잡한 패턴 학습
- 높은 수준의 추상화

### 일반 신경망과의 차이
- 복잡성: 더 많은 레이어와 파라미터
- 성능: 대규모 데이터셋에서 우수
- 학습 시간: 더 많은 시간과 자원 필요

## 3. 학습 알고리즘

### 3.1 순전파 알고리즘 (Forward Propagation)

#### 과정
1. 입력 데이터 수신
2. 가중치 곱셈 및 편향 덧셈
3. 활성화 함수 적용
4. 출력 레이어까지 반복

### 3.2 역전파 알고리즘 (Backpropagation)

#### 과정
1. 출력층에서 오차 계산
2. 각 뉴런의 가중치 영향 계산
3. 가중치 업데이트
4. 최적화를 위한 반복

## 4. 활성화 함수 (Activation Function)

### 정의
- 뉴런의 출력을 결정하는 비선형 함수

### 주요 함수
1. 시그모이드 (Sigmoid)
   - 범위: 0 ~ 1
   - 용도: 이진 분류

2. ReLU (Rectified Linear Unit)
   - 특징: 양수는 그대로, 음수는 0
   - 장점: 빠른 학습, 기울기 소실 완화

3. 탄젠트 쌍곡선 (Tanh)
   - 범위: -1 ~ 1
   - 특징: 시그모이드보다 넓은 출력 범위

4. Softmax
   - 용도: 다중 클래스 분류
   - 특징: 확률 분포 출력
